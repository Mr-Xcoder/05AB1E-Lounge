    05AB1E is a golfing language created in late 2015 by Programming Puzzles & Code Golf user Adnan. 05AB1E has always been particularly terse, and it excels when it comes to string-manipulation and compression.

What's the purpose of this site?

    The 05AB1E Lounge is supposed to be a place that eases the learning process and that helps people find important resources faster. And because the best way to learn is by practicing yourself, we're also working on a set of challenges for you to try out! If you have any questions, feel free to join us in 05AB1E's chatroom, and we'll happily review your code and help you improve it.
